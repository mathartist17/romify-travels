# Asset organization

Use these folders to keep the travel media tidy:

- `public/assets/brand/` for the main logo, icon mark, and alternate logo versions.
- `public/assets/destinations/meghalaya/` for Meghalaya images.
- `public/assets/destinations/arunachal-pradesh/` for Arunachal Pradesh images.
- `public/assets/destinations/uttar-pradesh/` for Uttar Pradesh images.
- `public/assets/customers/` for approved customer-uploaded photos.

Recommended naming:

- `logo-main.png`
- `logo-mark.png`
- `meghalaya-01.jpg`
- `meghalaya-02.jpg`
- `arunachal-01.jpg`
- `uttar-pradesh-01.jpg`

Keep image sizes compressed before adding them to the repository so the homepage stays light.
